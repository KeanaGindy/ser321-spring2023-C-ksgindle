You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=1&num2=2 -- returns multiplication of two numbers l
      • Sample url: http://34.218.233.62:9000/multiply?num1=2&num2=45
  • /github?query=users/amehlhase/repos -- returns public repos of users with full_name, ids, and login of the owner of each repo
      • Sample url: http://34.218.233.62:9000/github?query=users/amehlhase316/repos
  • /feedfishes?fishes=1&food=2 -- returns the amount of fishes fed, the amount of food given, and the type of fish fed
      • Sample url: http://34.218.233.62:9000/feedFishes?fishes=2&food=10
  • /tax?itemPrice=24&salesTax=7.25 -- returns total price of the item with the sales tax
      • Sample url: http://34.218.233.62:9000/tax?itemPrice=1.25&salesTax=10.2
